# Sizzle & Drizzle ‚Äì Restaurant Management System  

A full-stack **PHP & MySQL** powered restaurant management system designed to handle menus, orders, and customer interactions for **Sizzle & Drizzle**.  
This project demonstrates CRUD operations, database handling with PDO, responsive frontend with Bootstrap, and dynamic features like contact forms and image resizing.  

---

## Features  

- **Dynamic Menu Display** ‚Äì Browse food items stored in MySQL, with images and descriptions.  
- **Order System** ‚Äì Place food orders with contact & address validation.  
- **Image Upload & Resize** ‚Äì Upload menu images with auto-resizing into multiple formats (display, card, menu-card).  
- **Contact Form** ‚Äì Customers can send inquiries via email (PHP mail).  
- **Admin Panel (Manage Menu)** ‚Äì Add new dishes, descriptions, and prices.  
- **Responsive UI** ‚Äì Built with **Bootstrap 5**, Font Awesome, and Remix Icons.  
- **Secure Database Handling** ‚Äì PDO with prepared statements to prevent SQL injection.
